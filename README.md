# Proyecto JavaScript – Guía de Fundamentos (desde Python)

## 📌 Descripción

Este proyecto es una guía práctica para entender los fundamentos de JavaScript, especialmente pensada para personas que vienen de Python y ya dominan conceptos como variables, condicionales, ciclos, funciones y modularización.

El objetivo no es solo aprender sintaxis, sino **entender cómo piensa JavaScript**, ya que su modelo mental es diferente (y a veces confuso si no se aborda correctamente).

---

## 🎯 Objetivo del proyecto

* Comprender cómo funciona JavaScript en el navegador
* Adaptar tu forma de pensar de Python → JavaScript
* Dominar conceptos base que suelen causar confusión
* Practicar con ejercicios progresivos (básico → medio)

## 🧠 Temas cubiertos

### 1. Variables en JavaScript

* `var` → antigua, con hoisting (puede causar errores)
* `let` → moderna, mutable
* `const` → no permite reasignación

---

### 2. Hoisting

JavaScript “eleva” declaraciones antes de ejecutar el código.

👉 Puede generar comportamientos inesperados si no se entiende bien.

---

### 3. Closures

Funciones que **recuerdan variables aunque ya hayan terminado de ejecutarse**.

👉 Muy usado en:

* contadores
* encapsulación
* manejo de estado

---

### 4. Callbacks

Funciones que se pasan como argumento a otras funciones.

👉 Base de:

* eventos
* asincronía
* programación funcional

---

### 5. Pensamiento en JavaScript

A diferencia de Python:

| Python                      | JavaScript                 |
| --------------------------- | -------------------------- |
| Orientado a objetos clásico | Basado en funciones        |
| Flujo síncrono por defecto  | Mucha asincronía           |
| Tipado más claro            | Más flexible (y peligroso) |

## 🚀 Cómo usar este proyecto

1. Abre `index.html` en tu navegador
2. Lee la teoría y ejemplos
3. Abre `script.js`
4. Resuelve los ejercicios directamente ahí
5. Usa la consola del navegador (`F12`) para probar

## ⚠️ Recomendaciones importantes
