JavaScript desde Cero (Modo Realista)

Guía pensada para desarrolladores que vienen de Python

¿Qué es JavaScript realmente?

JavaScript es un lenguaje de programación interpretado que corre principalmente en el navegador, pero también en servidores (Node.js). A diferencia de Python, JavaScript está profundamente ligado al navegador y al manejo de interfaces.

Es importante entender algo desde el inicio: JavaScript puede volverse confuso rápidamente si no se entienden bien conceptos como funciones, contexto (this) y asincronía.

¿Por qué parece difícil?

  • No es orientado a clases tradicional (aunque ahora tenga clases)
  • Está basado en funciones
  • Maneja asincronía (promesas, callbacks)
  • El comportamiento de this cambia
  • Variables con reglas diferentes (var, let, const)

JavaScript es un lenguaje basado en funciones

En JS, las funciones son ciudadanos de primera clase. 

function saludar() {
  console.log("Hola");
}

const otra = saludar;
otra();

Puedes guardar funciones en variables, pasarlas como parámetros y retornarlas.

Objetos en JavaScript

Todo en JS gira alrededor de objetos. 

const persona = {
  nombre: "Moisés",
  edad: 25,
  saludar() {
    console.log("Hola " + this.nombre);
  }
};

persona.saludar();

A diferencia de Python, aquí los objetos son más flexibles pero también menos estrictos.

Variables: var vs let vs const

  • var: vieja, con hoisting raro
  • let: moderna, cambia valor
  • const: no cambia referencia

Ejercicios (Nivel Básico - Medio)

Básicos

  1. Crear una función que reciba 2 números y retorne la suma
  2. Crear una función que diga si un número es par o impar
  3. Convertir una lista de números en sus cuadrados
  4. Contar cuántas letras tiene una palabra
  5. Invertir un string

Intermedios

  1. Crear un objeto "usuario" con métodos
  2. Simular un contador usando closure
  3. Crear una función que reciba otra función (callback)
  4. Filtrar números mayores a 10 en un array
  5. Reducir un array a su suma total

Más avanzados

  1. Simular una promesa con setTimeout
  2. Crear una función async que consuma datos falsos
  3. Construir una clase Persona
  4. Extender la clase a Estudiante
  5. Manipular el DOM: crear un botón dinámico

Ejercicios DOM

  1. Crear un input y mostrar lo que escribe el usuario en tiempo real
